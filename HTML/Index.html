Modulos em JavaScript

Módulos são uma forma de organizar o código JavaScript em arquivos separados (02). Isto pode ser útil para tornar o código mais fácil de ler e manter.

add.js

main.js

index.html

O arquivo add.js exporta a função add, que pode então ser importado para o arquivo main.js. Isso nos permite manter nosso código organizado e mais fácil de ler. Lembre-se que para usar o recurso de modulos é preciso definir o atributo type do elemento script com o valor module no arquivo HTML5.
